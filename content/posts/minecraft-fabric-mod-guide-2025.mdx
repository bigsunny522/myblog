最近マイクラのMODを増やしたので、備忘録も兼ねてまとめておきます。

MODローダーには大きく分けてForgeとFabricがありますが、今回はFabricのほうを紹介します。

---

## 前提MOD

### 1. Fabric

MODを導入するために必要なものです。まず公式サイトから「Fabric Loader」をダウンロード・インストールします。

**公式サイト:** [Fabric 公式サイト](https://fabricmc.net)

個人的には、MOD導入時にゲームが壊れるのを防ぐために別ディレクトリを用意するほうが良いと思っているため、インストーラーの「起動構成を作成」チェックは入れずに進めています。

インストールが完了すると、指定したディレクトリ内に `mods` フォルダが作成され、そこにMODファイルを入れていく形になります。

### 2. Fabric API

FabricでMODを使用するために**必須**となるMODです。インストールしたFabricと同じバージョンのAPIをダウンロードし、`mods` フォルダに入れます。

配布先: [Fabric API (Modrinth)](https://modrinth.com/mod/fabric-api)

### 3. 起動方法

Fabricを使用して起動するには、起動構成の作成が必要です。

- **作成済みの場合:** Minecraft Launcherを起動すると自動的に選択されています。
- **新規作成する場合:** Launcher上部の「起動構成」タブから新規作成を選びます。

設定のポイントは以下の通りです。

| 項目 | 設定内容 |
| --- | --- |
| バージョン | `release fabric-loader-0.xx.x-1.xx.x` を選択 |
| ゲームディレクトリ | 任意の場所を指定。デフォルトは `C:\Users\ユーザー名\AppData\Roaming` |

筆者は `...\MinecraftMod` という別フォルダを作成して指定しています。

## シェーダー（影MOD）

影MODを使用するために必要な2つの基本ツールと、おすすめのシェーダーパックを紹介します。

### 必須ツール

**Iris Shaders**

公式サイトの「DOWNLOAD UNIVERSAL JAR」から導入します。

**公式サイト:** [Iris Shaders 公式サイト](https://irisshaders.dev)

配布先: [Sodium (Modrinth)](https://modrinth.com/mod/sodium)

## MOD管理ツール（Ferium）

MODが増えてくると、バージョン確認や一括アップデートが手間になってきます。**Ferium**はコマンドラインで動く軽量・高速なMODマネージャーです。Modrinthのプロジェクトをプロファイル単位でまとめて管理でき、`ferium upgrade` 一発で全MODを最新版にそろえられます。

### インストール

ターミナル（PowerShell またはコマンドプロンプト）を開いて以下を実行します。

```bash
winget install GorillaDevs.Ferium
```

インストール後はターミナルを再起動し、`ferium --version` が通ることを確認してください。

### プロファイルの作成

Feriumはプロファイル単位でMODを管理します。まず以下のコマンドでプロファイルを作成します。

```bash
ferium profile create
```

対話形式で以下の項目を入力します。

| 項目 | 入力内容 |
| --- | --- |
| Profile name | 任意の名前（例: `fabric-1.21`） |
| Minecraft version | 使用するバージョン（例: `1.21.1`） |
| Mod loader | `fabric` を選択 |
| Output directory | `mods` フォルダのフルパス |

`mods` フォルダのパスは、たとえば `C:\Users\ユーザー名\MinecraftMod\mods` のように指定します。

### MODの追加

Modrinthのページ URL（`modrinth.com/mod/<スラッグ>`）のスラッグ部分を指定して追加します。

```bash
ferium add fabric-api
ferium add sodium
ferium add jade
```

登録済みMODの確認は `ferium list` で行えます。

### ダウンロード・アップデート

登録したMODをすべて `mods` フォルダへダウンロードします。アップデートも同じコマンドです。

```bash
ferium upgrade
```

最新バージョンが自動で取得され、既存ファイルも上書きされます。

### その他の操作

| コマンド | 内容 |
| --- | --- |
| `ferium remove` | MODを対話形式で選択して削除 |
| `ferium profile list` | プロファイル一覧を表示 |
| `ferium profile switch` | 使用するプロファイルを切り替え |

複数のプロファイル（シェーダー用・建築用など）を作っておくと、環境の使い分けが簡単になります。

---

## まとめ

Fabricは軽量で導入がシンプルなため、最初のMOD環境としておすすめです。シェーダーだけでもビジュアルが大きく変わるので、まずはSodium + Iris + 好みのシェーダーから試してみると良いと思います。

MODを増やすときはバージョンの互換性に注意しつつ、少しずつ追加していくのが壊れにくくておすすめです。
